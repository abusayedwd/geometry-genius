Q. What is DOM and what is the purpose of DOM?

A. The Document Object Model is a programming API for HTML and XML documents. It defines the logical structure of documents and the way a document is accessed and manipulated.

Q. What is event bubble?

A.Event Bubbling is a concept in the dom. It happens when an element receives an event, and that event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element. The major use of event bubbling is the registration of default functions present in the program.

Q. What is a callback function and why will you use it?

A. the callback function in javascript is a function that is passed as an argument in another function. Which is then called inside the parent function to complete a routine or an action.
